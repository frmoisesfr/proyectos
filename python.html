📦 Variables y Tipos de Datos

Almacenando información

Básico

¿Qué es una variable?

Una variable es un contenedor que almacena datos. Piensa en ella como una caja etiquetada donde guardas información.

✅ Crear variables:
nombre = "Juan" edad = 25 altura = 1.75 es_estudiante = True

Tipos de Datos

📝 Tipos principales:
  • int: Números enteros (5, -10, 0, 1000)
  • float: Números decimales (3.14, -2.5, 9.99)
  • str: Texto ("Hola", 'Python', "2024")
  • bool: Verdadero o Falso (True, False)

Ver el Tipo de una Variable

numero_entero = 42 numero_decimal = 3.14 texto = "Python es genial" es_verdad = True print(type(numero_entero)) # print(type(numero_decimal)) # print(type(texto)) # print(type(es_verdad)) #

Nombrando Variables

Hay reglas para nombrar variables:

  • ✅ Deben empezar con letra o guion bajo (_)
  • ✅ Pueden contener letras, números y guiones bajos
  • ❌ No pueden contener espacios
  • ✅ Usa nombres descriptivos: edad es mejor que a

➕ Operadores

Cálculos y comparaciones

Básico

Operadores Aritméticos

a = 10 b = 3 print(a + b) # 13 - Suma print(a - b) # 7 - Resta print(a * b) # 30 - Multiplicación print(a / b) # 3.333 - División print(a // b) # 3 - División entera print(a % b) # 1 - Módulo (residuo) print(a ** b) # 1000 - Potencia

Operadores de Comparación

Devuelven True o False:

x = 5 y = 10 print(x == y) # False - igualdad print(x != y) # True - desigualdad print(x < y) # True - menor que print(x > y) # False - mayor que print(x <= y) # True - menor o igual print(x >= y) # False - mayor o igual

Operadores Lógicos

edad = 25 tiene_carnet = True # AND - ambas condiciones deben ser verdaderas print(edad >= 18 and tiene_carnet) # True # OR - al menos una condición debe ser verdadera print(edad < 18 or tiene_carnet) # True # NOT - invierte el valor print(not (edad < 18)) # True

⌨️ Entrada de Datos (Input)

Pedir datos al usuario

Básico

¿Qué es Input?

La función input() permite que el usuario escriba datos desde el teclado. El programa espera a que escriba algo y presione Enter.

Sintaxis Básica

nombre = input("¿Cuál es tu nombre? ") print(f"Hola, {nombre}!")

⚠️ Importante: input() siempre devuelve un string. Si necesitas un número, debes convertirlo.

Convertir a Números

edad_texto = input("¿Cuántos años tienes? ") edad = int(edad_texto) # Convertir a entero print(f"Tienes {edad} años") print(f"El año que viene tendrás {edad + 1}")

También Puedes Hacerlo en Una Línea

# Convertir directamente edad = int(input("¿Cuántos años tienes? ")) peso = float(input("¿Cuánto pesas en kg? ")) print(f"Edad: {edad}, Peso: {peso}kg")

Múltiples Datos de Entrada

nombre = input("Nombre: ") apellido = input("Apellido: ") edad = int(input("Edad: ")) ciudad = input("Ciudad: ") print(f"\n=== DATOS ===" ) print(f"Nombre completo: {nombre} {apellido}") print(f"Edad: {edad} años") print(f"Ciudad: {ciudad}")

✅ Ejemplo 2: Calcular IMC (Índice de Masa Corporal)
nombre = input("¿Cuál es tu nombre? ") peso = float(input("¿Cuál es tu peso en kg? ")) altura = float(input("¿Cuál es tu altura en metros? ")) imc = peso / (altura ** 2) print(f"\n{nombre}, tu IMC es: {imc:.2f}") if imc < 18.5: print("Peso bajo") elif imc < 25: print("Peso normal") elif imc < 30: print("Sobrepeso") else: print("Obesidad")
✅ Ejemplo 3: Ficha de Registro
